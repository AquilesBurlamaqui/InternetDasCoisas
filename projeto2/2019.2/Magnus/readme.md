Iniciamos instalando o DroidScript, aplicativo que possui várias referências de códigos em JavaScript, como exemplos com sensor de luz ou acelerômetro. No caso, usamos o código de sensor de luz para captar os valores de luz capturados pelo sensor do célular. Enviando esses valores para o ThingSpeak para que exiba as informações a cada 10 segundos, observando o comportamento desse processo em nossa página, a qual hospedamos no githubpages ou outa plataforma, como o 000webhost.
